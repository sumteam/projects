* `DD/MM/YYYY HH:mm:ss`
* `MM/DD/YYYY HH:mm:ss`

### **Prediction Columns**

Prediction columns must follow this pattern:

```

```

Where:

* **number** = any integer
* **unit** =

  * `s` â†’ seconds
  * `m` â†’ minutes
  * `h` â†’ hours

#### **Examples**

```

```

### **Prediction Values**

* `1` â†’ positive causal chain detected
* `0` â†’ no chain detected
